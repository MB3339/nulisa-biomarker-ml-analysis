# NULISA Biomarker ML Analysis

This repository contains a machine learning workflow for analyzing high-dimensional biomarker data generated using the NULISA platform.

## Project Overview
The goal of this project is to explore biological patterns across biomarker datasets using supervised and unsupervised machine learning approaches. The workflow includes:

- data preprocessing
- missing value handling
- feature scaling
- dimensionality reduction (PCA)
- clustering analysis
- supervised classification

## Tools and Libraries
- Python
- pandas
- NumPy
- scikit-learn
- matplotlib
- seaborn

## Project Structure
- `src/nulisa_ml_analysis.py`: main analysis pipeline
- `notebooks/`: exploratory notebook workflows

## Notes
This repository is intended to demonstrate my approach to organizing and analyzing large biological datasets using reproducible machine learning workflows.
